Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

Image Tags & CSS Selectors

Attribute Name Description
<img> Image in HTML Define src either through URL or folder structure
background-image Background Image define via a url() path.

CSS Positions

Attribute Name Description
static Static Position Default position for an element.
relative relative Position Relative to its normal position. When using top/bottom/left/right it is putting the element away from that position
fixed fixed Position Will always stay in the same place, even when the page is scrolled. The top/right/bottom/left properties are used to position the element.
absolute absolute Position Position relative to the nearest positioned ancestor compared to being positioned relative to the viewport.
sticky sticky Position Positioned based on the user's scroll position. Stays relative until scrolled past and then becomes fixed.

Breadcrumbs

Attribute Name Description
.. Location Location of where you are on the site and past locations.
.. Attribute Similar to location, but an attribute of your location. Example: Hotels -> Queen Room // Pets - Beach view
.. Path-based Based on where the user has been on the site. Custom for each user. Try to avoid.

Breadcrumbs are created via a <ul></ul> list and includes the class breadcrumbs. Each <li></li> will have a href="#" with # being the location or # if that is the current location

Example Below:

        
          <ul class="breadcrumb">
            <li><a href="shop.html">Shop</a></li>
            <li><a href="groceries.html">Groceries</a></li>
            <li><a href="blueberries.html">Blueberries</a></li>
            <li><a href="#">Organic</a></li>
          </ul>
